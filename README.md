# LumenQuery

Enterprise-grade [Stellar](https://stellar.org) Horizon API and Soroban RPC infrastructure.

**Website:** [https://lumenquery.io](https://lumenquery.io)

## What is LumenQuery?

LumenQuery provides fast, reliable API access to the Stellar blockchain network. Built for developers, traders, validators, and institutions.

### Features

- **[Horizon API](https://lumenquery.io/docs)** - Full access to accounts, transactions, payments, and ledger data
- **[Soroban RPC](https://lumenquery.io/docs/contracts)** - Deploy and interact with smart contracts
- **[Network Analytics](https://lumenquery.io/analytics)** - Real-time Stellar network metrics (public, no auth required)
- **[Smart Contract Explorer](https://lumenquery.io/contracts)** - Decoded Soroban contract calls, storage, and events
- **[Live Transactions](https://lumenquery.io/dashboard/transactions)** - Real-time transaction stream with decoded operations
- **[Natural Language Query](https://lumenquery.io/query)** - Query the Stellar blockchain in plain English
- **[Portfolio Intelligence](https://lumenquery.io/portfolio)** - Multi-account aggregation, P&L tracking, yield monitoring

### Tech Stack

- **Frontend**: Next.js, TypeScript, Tailwind CSS
- **Backend**: Node.js, Fastify, PostgreSQL, Redis
- **Infrastructure**: Docker, Traefik, Prometheus, Grafana
- **Blockchain**: Stellar Captive Core, Soroban RPC
